# 🎓 Student Exam Performance Analyzer

A Python-based project that analyzes student exam performance and provides meaningful insights such as subject-wise percentages, overall performance, and identification of strengths and weaknesses.


1) Overview:

This project is designed to help evaluate a student's academic performance using basic data analysis techniques. It processes subject-wise marks and generates useful insights that can assist in academic improvement.


2) Features

* Calculates percentage for each subject
*  Computes overall percentage
*  Identifies highest scoring subject
*  Identifies lowest scoring subject
*  Uses structured data handling with Pandas


3) Technologies Used

* **Python**
* **Pandas**
* **Google Colab / Jupyter Notebook**


4)  Project Structure

* `STUDENT_EXAM_PERFORMANCE_ANALYZER.ipynb` → Main notebook containing code and analysis
* `README.md` → Project documentation


5)  How to Run the Project

1. Open the notebook in Google Colab
2. Run all cells step by step
3. View the calculated results and analysis


6)  Sample Output

* Subject-wise percentages
* Overall percentage
* Best performing subject
* Weakest subject


7)  Purpose of the Project:

This project was developed as part of learning **data analysis fundamentals using Python**. It demonstrates the ability to:

* Work with real-world structured data
* Perform calculations and analysis
* Present meaningful insights
